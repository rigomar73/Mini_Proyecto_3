DESCRIPCIÓN DEL PROYECTO
------------------------
Este proyecto consiste en una simulación de batallas Pokémon utilizando el lenguaje de programación Java.  
Incluye la gestión de entrenadores, Pokémon, movimientos y lógica de combate.  

INSTRUCCIONES DE EJECUCIÓN
--------------------------
1. Clonar o descargar el repositorio.  
2. Abrir el proyecto en el entorno de desarrollo.  
3. Ejecutar la clase `Main.java` ubicada en la carpeta `src/`.
